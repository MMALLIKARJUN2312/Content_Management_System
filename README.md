# RenewCred CMS

Headless CMS for the RenewCred "Standards" section, plus the public frontend that
consumes it. Monorepo: `backend` (Express + MongoDB), `admin-frontend` (Next.js + Redux
Toolkit), `public-frontend` (Next.js).

> Full documentation (setup, architecture, assumptions) lands in a later phase of this
> build. This is a placeholder so the repo isn't undocumented in the meantime.
